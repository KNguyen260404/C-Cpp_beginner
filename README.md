## ⚡ Cài Đặt Nhanh

### 🖥️ Windows (MinGW/MSYS2)
```bash
# Cài đặt MinGW từ https://sourceforge.net/projects/mingw/
g++ -std=c++17 -o calculator.exe 01_calculator.cpp
calculator.exe
```

### 🐧 Linux Ubuntu/Debian
```bash
# Cài đặt g++
sudo apt update && sudo apt install g++

# Biên dịch và chạy
g++ -std=c++17 -o calculator 01_calculator.cpp
./calculator
```

### 🍎 macOS
```bash
# Cài đặt Xcode Command Line Tools
xcode-select --install

# Hoặc sử dụng Homebrew
brew install gcc

# Biên dịch và chạy
g++ -std=c++17 -o calculator 01_calculator.cpp
./calculator
```
